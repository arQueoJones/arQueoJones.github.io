arqueobot: suite completa de inteligencia arqueológica

Gratuita para uso no comercial. Integra diseño de museos, detección de fragmentos, GPT4All, fotogrametría y mucho más.

¿Qué es arqueobot?

Un ecosistema Python de código abierto que reúne las herramientas más avanzadas del proyecto WAAZP: desde el entrenamiento de clasificadores hasta la simulación de flujos de visitantes en museos.

Módulos principales

Museum AI Builder, detección de shards, asistente conversacional (GPT4All), preprocesado de imágenes, análisis estadístico multivariante, integración COLMAP, y sistema de categorías jerárquicas.

Acceder a GitHub y documentación →

Características destacadas

  • 100% local, sin necesidad de conexión a internet
  • Interfaz gráfica amigable (tkinter)
  • Soporte para múltiples idiomas (ES, EN, FR, DE)
  • Licencia MIT / investigación · Comercial bajo petición

Nota: arqueobot es el núcleo técnico que impulsa ar.QueoJones. Puedes descargarlo, ejecutarlo y modificarlo libremente para tus proyectos académicos.
